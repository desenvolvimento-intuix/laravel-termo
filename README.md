# laravel-termo

Pacote laravel de Termo...

# Instalação

 1. Editar o arquivo composer.json adicionando os seguintes parâmetros:

 ```php
   {
	    "repositories": [
	    {
		    "type": "vcs",
		    "url": "https://github.com/desenvolvimento-intuix/laravel-termo"
	    }
    ],
	    "require": {
		    "intuix/laravel-termo": "dev-master"
	    }
    }
```
